# fasm

## Links

FlatAssembler https://flatassembler.net/


## Install

```bash
wget https://flatassembler.net/fasm-1.73.32.tgz
tar -xzf fasm-1.73.32.tgz
sudo mv fasm /usr/bin/
echo 'export PATH=$PATH:/usr/bin/fasm' >> ~/.bashrc && source ~/.bashrc
```
